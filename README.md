# 🍽️ Application de Recettes de Cuisine

## 📌 Description
Application web pour gérer des recettes de cuisine avec catégories, recherche et favoris.

---

## ⚙️ Technologies utilisées
- Angular 17
- Node.js + Express
- MongoDB + Mongoose
- SweetAlert2

---

## 🚀 Fonctionnalités

### ✔ Recettes
- Ajouter une recette
- Modifier une recette
- Supprimer une recette
- Afficher les recettes
- Détail d’une recette
- Recherche par nom
- Filtrage par catégorie et difficulté
- Pagination

### ✔ Catégories
- CRUD catégories
- Affectation à une recette

### ✔ Utilisateur
- Login
- Favoris




- SweetAlert2 (notifications modernes)
- Interface responsive
- Pagination
- Favoris utilisateur
